# NEU Framework Pilot Study: Burnout Domain Analysis

## Executive Summary
This repository contains the R-based data analysis for the initial pilot of the NEU Framework, a behavioral science intervention designed to quantify and mitigate occupational burnout. Developed by JAI Behavioural, this pilot study evaluates the efficacy of the framework by analyzing pre- and post-intervention data from a cohort of educational professionals. 

The analysis focuses on extracting actionable insights regarding how specific domains of burnout respond to targeted behavioral interventions over a two-week period.

## Methodology & Scope
[cite_start]The data examines "NEU Pilot 1 – All Completers" [cite: 4] [cite_start]by tracking metric shifts from "Day 0 to Day 14"[cite: 3]. [cite_start]The assessment framework utilizes a "Mean Score (/9)" [cite: 1] [cite_start]to quantify symptoms across four primary areas of the "Burnout Domain"[cite: 12]:
* [cite_start]**Emotional** [cite: 8]
* [cite_start]**Motivation** [cite: 9]
* [cite_start]**Relational** [cite: 10]
* [cite_start]**Cognitive** [cite: 11]

[cite_start]Additionally, the analysis tracks the "TBP Total Score (/36)" [cite: 30] to measure holistic burnout profile changes at the individual participant level. 

## Key Findings
The visualizations generated by this analysis (`Domainmeanv1.pdf` and `TBPscoresv1.pdf`) demonstrate quantifiable improvements in the cohort's well-being following the intervention.

**1. Domain-Level Impact**
* [cite_start]The intervention yielded positive aggregate results, evidenced by a visible reduction in the "Mean Score (/9)" [cite: 1] [cite_start]across every measured "Burnout Domain"[cite: 12].
* [cite_start]Scores decreased from Day 0 (Navy) to Day 14 (Orange) [cite: 13] [cite_start]uniformly across the "Emotional" [cite: 8][cite_start], "Motivation" [cite: 9][cite_start], "Relational" [cite: 10][cite_start], and "Cognitive" [cite: 11] categories.

**2. Individual Efficacy (TBP Score)**
* [cite_start]An analysis of the "TBP Total Score Change: Day 0 to Day 14" [cite: 23] highlights significant individual-level improvements.
* [cite_start]The distinct majority of the tracked cohort—specifically participants P4 [cite: 14][cite_start], P9 [cite: 15][cite_start], P8 [cite: 16][cite_start], P6 [cite: 17][cite_start], P5 [cite: 18][cite_start], P7 [cite: 19][cite_start], and P2 [cite: 22][cite_start]—demonstrated a reduction in their overall burnout scores from Day 0 to Day 14[cite: 23, 24]. 
* [cite_start]Participants P1 [cite: 21] [cite_start]and P3 [cite: 20] exhibited slight score increases, providing crucial variance data for future refinement of the framework's targeting mechanisms.

## Strategic Implications
This pilot validates the NEU framework as a viable, data-driven tool for organizational burnout prevention. By segmenting burnout into distinct cognitive and emotional domains rather than treating it as a monolithic condition, interventions can be deployed with greater precision. The R scripts provided in this repository establish the foundation for scaling this analytics pipeline for broader institutional audits.
